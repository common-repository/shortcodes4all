Enhance your theme with beautiful shortcodes. With Shortcodes4all you can easily create buttons, warning boxes, columns, testimonials, etc.

== Description ==

Shortcodes are simple commands written directly to the post or page text area. They are enclosed in square brackets – **[shortcode]text[/shortcode]** – and can have one or more parameters. We have prepared them to make your life easier. With our shortcode plugin you can easily create buttons, warning boxes, columns, etc. using shortcodes.

With this plugin you can create following shortcodes:

* Column Shortcodes
* Content Box Shortcodes
* Boxes Shortcodes
* Buttons Shortcodes
* Toggle content
* Tooltip text
* Author information
* Custom list
* Testimonials
* Tabgroup
* Login form
* Price tables
* Quotations
* Table

*Please visit the documentation page for a full description: [Shortcodes4all](http://shortcodes.themes4all.com/)*

== Installation ==

*Using FTP*

1. Upload `shortcode4all` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

or

*Using The WordPress Dashboard*

1. Navigate to the 'Add New' in the plugins dashboard
1. Search for 'Shortcodes4all'
1. Click 'Install Now'
1. Activate the plugin on the Plugin dashboard
